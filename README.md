# Servidor de Scraping para Notificaciones Legales

Este servidor proporciona una API para obtener notificaciones legales de diferentes servicios gubernamentales argentinos (AFIP, PJN, TAD) mediante técnicas de web scraping.

## Requisitos

- Node.js 16.x o superior
- npm 7.x o superior
- Dependencias del sistema para Puppeteer (ver sección de instalación)
